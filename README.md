# Telegram Summary

Инструмент для создания кратких summary из экспортированных диалогов Telegram.

Проект создаётся как pet project / портфельный проект:
- изучение работы с LLM
- работа с длинными диалогами
- проектирование client / server архитектуры

## Зачем он нам нужОн этот ваш TG Summary
Telegram-чаты быстро разрастаются и становятся неудобными для анализа.
Этот инструмент помогает:
- получить краткое содержание диалога
- выделить ключевые темы
- зафиксировать итоги обсуждений

## Архитектура
Проект состоит из двух частей:

- **client** — open-source клиент (CLI), доступен в этом репозитории
- **server** — серверная часть для работы с LLM (может быть закрытой)

## Статус проекта
Ранняя стадия разработки (alpha).
Функциональность будет расширяться.
